Docker数据卷

  • 数据卷用途：使容器内部的文件跟容器外面的文件同步
  • 数据卷的特点:
    1. 数据卷可在容器之间共享或重用数据
    2. 卷中的更改可以直接生效
    3. 数据卷中的更改不会包含在镜像的更新中
    4. 数据卷的生命周期一直持续到没有容器使用它为止
